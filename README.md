# Skilaverkefni3

-NBA Quiz 2026-
Stutt og skemmtilegt quiz forrit byggt í React og TypeScript. Notandinn getur svarað spurningum um NBA deildina, stillt upplifunina sína og skoðað svörin sín í lokin.

-Hvernig á að keyra verkefnið-
Fylgdu þessum skrefum til að keyra verkefnið á tölvunni þinni:

Setur þessa skipun í terminal:
npm install

Síðan seturu þessa skipun í terminal:
npm run dev
Eftir þetta ættir þú að geta opnað síðuna http://localhost: í vafranum.

-Virkni síðunar-
Forritið inniheldur alla grunnvirkni sem beðið var um:

Spurningakerfi: Notandi getur valið svör og flakkað á milli spurninga.

Stigatöfla: Sýnir lokastig í lokin.

Leiðbeiningar: Upphafsskjár sem útskýrir leikinn.

Hætta í quizi: Hægt er að hætta í leiknum hvenær sem er með X uppi í hægra horni.

-Auka-áskoranir-
Ég bætti við eftirfarandi aukavirkni til að gera forritið öflugra:

Stillingar (Settings): Notandi getur kveikt/slökkt á eiginleikum áður en leikurinn hefst.

Tímamælir: Hægt er að hafa teljara sem sýnir hversu lengi var verið að svara.

Progress Bar: Sjónræn stika sem fyllist eftir því sem líður á quizið.

Instant Feedback: Valkvæð stilling til að sjá rétt/rangt svar um leið og smellt er.

Review Mode: Eftir að quizið er búið getur notandinn smellt á hringina til að skoða hverja spurningu og séð hvað var rétt.

Confetti: Notaði canvas-confetti til að fagna þegar quizið er klárað.

Email Link: Tákn neðst á forsíðu til að senda póst.

-Auka upplýsingar-
Í app.css og app.tsx skránni ákvað ég að prófa að flokka allt svo það myndi auðveldi mér að finna og laga villur og fyrir ykkur að lesa
